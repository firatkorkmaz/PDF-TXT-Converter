# PDF to TXT Converter
A simple program to convert multiple PDF files to TXT files by using **PyPDF2** library.

## General Information
This program converts PDF files to TXT files by asking the user to enter either PDF filename(s) or PDF folder name from the input and saves the PDF contents in text files.
	
## Technologies
This project is created with:
* Python
* Jupyter Notebook

## Setup & Run
Although an automatic installation command is placed in the source code for the **PyPDF2** library, it can also be manually installed by:
```
pip install PyPDF2
```
There are 2 different files to run the program in different ways:
1. **PDF_TXT_Converter.ipynb** can be run with Jupyter Notebook:
```
pip install jupyter
```
```
jupyter notebook
```
2. **PDF_TXT_Converter.py** can be run directly with Python:
```
python PDF_TXT_Converter.py
```
There are also 2 PDF files given as input file examples: **file1.pdf** and **file2.pdf** with their saved results: **file1.txt** and **file2.png**
